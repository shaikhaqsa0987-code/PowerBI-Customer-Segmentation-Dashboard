# Power BI Customer Segmentation & Retention Dashboard 📊

This project is part of my Data Analytics learning journey.

## 📌 Objective
To analyze customer behavior, identify high-value and at-risk customers, and support retention strategies using data.

## 🔍 Key Features
- Customer segmentation by value and behavior
- Repeat vs one-time customer analysis
- Revenue contribution by segment
- Interactive slicers and filters

## 🛠 Tools & Technologies
- Power BI Desktop
- DAX
- Data Modeling
- Data Visualization

## 📈 Outcome
The dashboard provides insights into customer retention and helps businesses focus on high-impact customer segments.
